## ‚ú® Features

- **Work Sessions**: Default 25-minute work sessions
- **Break Sessions**: Default 5-minute break sessions  
- **Customizable Durations**: Adjust work and break durations to your preference (1-60 minutes for work, 1-30 minutes for breaks)
- **Timer Controls**: Start, pause, and reset the timer
- **Automatic Switching**: Automatically switches between work and break sessions
- **Visual Themes**: Different color themes for work mode and break mode
- **Responsive Design**: Works on desktop and mobile devices

## üéØ How It Works

1. Click **Start** to begin your work session
2. The timer counts down from 25 minutes (or your custom duration)
3. When the work session ends, it automatically switches to a break session
4. After the break, it returns to a work session
5. Use **Pause** to temporarily stop the timer
6. Use **Reset** to return to the initial work duration
